# SF_E6_Messager
Skill factory E6.9 task - project
MESSAGER

Usage:
1. Type command "cd messager"
2. Type command "pip install -r requirements.txt"
3. Use Redis server on address 127.0.0.1, port 6379.
4. Type command "python manage.py runserver"
